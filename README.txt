Análisis de Tarifas de Servicios Públicos – EPM

Proyecto de análisis de datos enfocado en el estudio de tarifas de servicios públicos (acueducto y alcantarillado) en municipios de Antioquia, utilizando datos públicos.

El objetivo es identificar patrones, variaciones y generar insights que faciliten la comprensión del comportamiento de las tarifas.

Enfoque del proyecto:

-Limpieza y transformación de datos

-Análisis exploratorio (EDA)

-Identificación de patrones y tendencias

-Generación de insights accionables

-Visualización de datos en dashboard interactivo

Herramientas utilizadas:
-SQL
-Power BI 
-Excel
Estructura del repositorio

/base de datos → Dataset y archivos relacionados con la fuente de datos

/dashboards → Archivo de Power BI con visualizaciones

/Etapas del Proyecto de Análisis epm → Documento explicativo del proyecto (metodología, análisis e insights)

Resultados:

Se lograron identificar diferencias relevantes en las tarifas entre municipios, así como patrones que permiten entender mejor el comportamiento del servicio.

Guia de usuario:

Explorar el dataset en la carpeta /base de datos
Revisar el proceso y conclusiones en /Etapas del Proyecto de Análisis epm
Abrir el dashboard en /dashboards para visualizar los resultados
